- install sass in local: npm install node-sass --save-dev 

- script compile sass to css: 
    "scripts": {
    "compile:sass": "node-sass sass/main.scss css/style.css -w"  // compile file main.scss thành code css và ghi nó vào file style.css, flag -w để lắng nghe => chỉ cần compile 1 lần và nó sẽ lắng nghe thay đổi
  },
  => để chạy, ấn lệnh npm run compile:sass

- live-server : npm install live-server -g // install global để bất cứ prj nào cx dùng dc
 => để chạy, gõ lệnh live-server trong folder main prj đó

- folder base in scss:
    + base : chứa các base scss như animation, typography.utilities và 1 file base
    + components: chứa các components để reuse
    + layout: chứa các file layout như footer,header,...
    + pages: chứa các file page như landing page, homepage
    + abstracts : chứa các file mixin, function, variable
    + vendor: chứa các file từ bên thứ 3 như bootstrap, animation framework,...
    + themes: chứa các file themes ví dụ khi muốn giao diện web riêng và web mobile riêng
